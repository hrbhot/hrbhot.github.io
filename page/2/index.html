Mdada创建raid1分区

2019-10-27
用fdisk 把硬盘的分区都删除。 树形列出硬盘和分区 lsblk -o NAME,SIZE,FSTYPE,TYPE,MOUNTPOINT 创建为md0的 raid1 分区 sdb/sdc是第二块和第三块硬盘 sudo mdadm –create –verbose /dev/md0 –level=1 –raid-devices=2 /dev/sdb /dev/sdc 格式化分区 mkfs.ext4 -F /dev/md0 挂载分区 mkidr -p /data mount /dev/md0 /data 检测分区是否已经正常挂载 df -h -x devtmpfs -x tmpfs 保存自动加载 mdadm配置 mdadm –detail –scan | sudo tee -a /etc/mdadm/mdadm.conf 加载到ram里面，在引导可用 update-initramfs -u 添加到启动以便重启自动加载 echo ‘/dev/md0 /mnt/md0 ext4 defaults,nofail,discard 0 0’ | sudo tee -a /etc/fstab 参考来源 https://www.
